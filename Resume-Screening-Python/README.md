# Resume Screening App

Automated resume screening system using NLP and Machine Learning to classify and rank candidate resumes based on job requirements. Saves HR time and improves recruitment accuracy.

## Features
- Upload resumes in PDF or DOCX format
- Automatically analyze and rank candidates
- Filter candidates based on skills and experience
- User-friendly Streamlit interface

## How to Use
1. Open the app on Hugging Face: [Your Space Link](https://huggingface.co/spaces/RISHA1811/resume-screening)
2. Upload your resume(s)
3. Select the job requirements or skills you want to match
4. View ranked candidate list
